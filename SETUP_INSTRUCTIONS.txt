QUICK START (DEVELOPMENT)
=========================

1. Navigate to project directory:
   cd /sessions/busy-confident-ptolemy/book-project/site

2. Install dependencies:
   npm install

3. Start development server:
   npm run dev

4. Open in browser:
   http://localhost:3000

5. Press Ctrl+C to stop server

PRODUCTION BUILD
================

1. Build the application:
   npm run build

2. Start production server:
   npm start

3. The site runs on port 3000 (or $PORT environment variable)

DEPLOYMENT OPTIONS
===================

OPTION 1: VERCEL (RECOMMENDED - EASIEST)
----------------------------------------
Vercel is the company behind Next.js and provides seamless integration.

1. Install Vercel CLI:
   npm install -g vercel

2. Deploy to Vercel:
   vercel --prod

3. Follow the prompts and your site will be live!

Alternative: Connect your Git repository to Vercel dashboard for automatic deployments.

OPTION 2: SELF-HOSTED (DOCKER)
------------------------------

1. Build Docker image:
   docker build -t answer-53 .

2. Run the container:
   docker run -p 3000:3000 answer-53

3. Visit http://localhost:3000

OPTION 3: SELF-HOSTED (NODE.JS)
-------------------------------

1. Build:
   npm run build

2. Start:
   npm start

3. The server listens on port 3000

4. For production process management, use PM2:
   npm install -g pm2
   pm2 start "npm start" --name "answer-53"
   pm2 save
   pm2 startup

OPTION 4: CLOUD PLATFORMS
--------------------------

AWS Lambda:
  Use Vercel deployment or AWS Amplify console

Google Cloud Run:
  1. Create Docker image
  2. Push to Google Container Registry
  3. Deploy to Cloud Run

Azure App Service:
  1. Create resource group and app service plan
  2. Deploy from GitHub or via Azure CLI

DigitalOcean:
  Deploy via App Platform or Droplet with Node.js

See DEPLOYMENT.md for detailed instructions for each platform.

CONFIGURATION
=============

Environment Variables (.env.local):
  NODE_ENV=production
  NEXT_PUBLIC_SITE_URL=https://your-domain.com

Modify Content:
  Edit lib/chapters.ts to change chapter content

Modify Colors:
  Edit tailwind.config.js colors.lcars section

Modify Fonts:
  Edit app/layout.tsx to import different Google Fonts

Add Analytics:
  Set NEXT_PUBLIC_GA_ID in .env.local for Google Analytics

TROUBLESHOOTING
===============

Port 3000 already in use:
  PORT=3001 npm run dev

Module not found errors:
  Delete node_modules and package-lock.json
  Run npm install again

Build fails:
  npm run build
  Check console for specific error messages

Styles not loading:
  npm run build
  Restart development server

TypeScript errors:
  These are caught during build
  npm run build shows all errors
